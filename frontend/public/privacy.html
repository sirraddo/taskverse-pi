TaskVerse Pi Privacy Policy

Last updated: June 2026

1. Who We Are

TaskVerse Pi is a decentralised micro-task marketplace built for the Pi Network ecosystem, accessible via the Pi Browser.

2. Data We Collect

  • Pi username — provided by the Pi Network SDK on login
  • KYC status — a verified/unverified flag from the Pi SDK
  • Task submissions — proof text and optional screenshot URLs you submit
  • Payment records — Pi payment identifiers for processing payouts

3. How We Use Your Data

  • To authenticate you via the Pi Network SDK
  • To process task payouts through Pi's A2U payment system
  • To detect and prevent fraudulent submissions
  • To display your earnings history and rank on the leaderboard

4. Data Sharing

We never sell your personal data. Data is shared only with:

  • Pi Network — for authentication and payment processing
  • ImgBB — for optional screenshot uploads in task submissions
  • MongoDB Atlas — for secure data storage

5. Data Retention

Account and submission data is retained for as long as you use the app. You may request deletion by contacting us.

6. Security

All data is transmitted over HTTPS. We use short-lived JWT tokens for session authentication. We do not store Pi private keys or seed phrases.

7. Children

TaskVerse Pi is not intended for users under 18. Use requires a verified Pi Network Pioneer account.

8. Contact

For privacy questions, reach out via the Pi Browser messaging system or through our Pi Developer portal listing.
